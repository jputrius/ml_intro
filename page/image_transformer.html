Image classification using transformers

Back to the main page.

Transformers have been shown to outperform CNNs in image classification on some large datasets.

Build a transformer model for image classification. For inspiration you can use this paper.

Task steps:

  • 1. Describe the model architecture you are going to use.
  • 2. Implement your model in PyTorch.
  • 3. Compare your model's performance on FashionMNIST, CIFAR10 and CIFAR100 datasets with the performance of a fine-tuned ResNet (compute accuracy, precision and recall).

Note that your model is probably not going to outperform ResNet on these small datasets. This is expected. Don't worry, you will not get a reduction in your grade.

FashionMNIST, CIFAR10 and CIFAR100 datasets ship with torchvision. You can find them here. Pretrained ResNet also ships with torchvision. You can find different versions here.
